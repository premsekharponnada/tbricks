Detailed Description

TradingLimitRule.

The TradingLimitRule object represents a single limit rule, each of which is checked for each transaction.

A TradingLimitRule object has two parts, "Filter" and "Limit", available via the corresponding GetFilter() and GetLimit() accessors. The filter defines which transactions will be checked by this rule based on conditions. The limit part represents the condition that will be evaluated for an order.

From the viewpoint of a limit rule provider, there is a difference between instrument-based rules; those that have an instrument as part of the filter, and "general" rules; those that do not have an instrument as a part of the filter
