Detailed Description

StrategyByParentFilter matches strategy instances which parent identifier (and, optionally, strategy type) matches the one set in the filter.

The filter therefore matches all direct children of the specified strategy (if SetWithParent() is set to "true", the filter also matches the parent strategy itself).
