Detailed Description

ISO-10962

The Classification of Financial Instruments (CFI) is used to describe financial instruments.

The CFI consists of 6 alphabetical characters.

1 character - the highest level of category, 2 character - specific groups, 3 - 6 characters - the most important attributes of each group.

Character 'X' used if information is unknown.

Support 2 standards : ISO_10962_2001, ISO_10962_2015 If standard is not specified, 2001 standard is used.
