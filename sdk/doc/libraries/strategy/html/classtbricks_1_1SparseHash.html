Detailed Description

template<typename KeyT, typename ValueT>
class tbricks::SparseHash< KeyT, ValueT >

SparseHash template.

SparseHash is very memory efficient and have reasonably good execution time for all operations. If table is not on a super critical path, SparseHash should be always preferred to Hash table (i.e. dense hash table).
