Detailed Description

CFIMask is a set of CFIs. It may be useful when you want to match some set of CFIs, for example, all options (OXXXXX, HXXXXX) or warrants (RWXXXX, RFXXXX). It is possible to create custom mask from vector of CFIs. Character 'X' in passed CFI used if information may be any.
