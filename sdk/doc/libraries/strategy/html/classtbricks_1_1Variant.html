Detailed Description

The class Variant represents a type-safe union of AnyType classes. An instance of Variant at any given time either holds a value of one of its alternative types, or be in empty, or remove state.

Supported types: PortfolioIdentifier, InstrumentIdentifier, Price, Volume, String, Double, SmartDouble, Boolean, Side, Venue, Integer, Enumeration, DateTime, Date, Time, TradeIdentifier, Mic, CFI, Currency, Duration, Identifier, InstrumentGroupIdentifier Table, Binary, GridSelection, CalculatedProperty, CalculatedPropertyFilter, TreeNodeIdentifier LiquidityPoolIdentifier, SubsystemIdentifier, LegalEntityIdentifier, TreeTypeIdentifier MMTFlags, BinaryTable, RankingIdentifier
