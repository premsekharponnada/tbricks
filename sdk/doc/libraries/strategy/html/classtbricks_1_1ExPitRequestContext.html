Detailed Description

This class is the primary pre-trade risk check point. When IStrategyEventHandler::HandleAppRequest() is called, ExPitRequestContext object contains parameter values extracted from the request, type of the request being handled, request identifier, strategy identifier, etc. Callee shall check and decide whether request is accepted/rejected for further processing.
