Detailed Description

PositionByAncestorFilter matches positions that have strategy identifier provided as a filter argument in their inheritance tree. 

      S0
     /  \
    S1  S2
   / |  | \
 S3 S4  S5 S6

If a strategy open stream with the identifier of S1, it should see any positions generated by S1 or its child apps.
