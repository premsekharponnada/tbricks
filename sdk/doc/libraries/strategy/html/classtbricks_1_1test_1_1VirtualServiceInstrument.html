Detailed Description

VirtualServiceInstrument provide possibility to work with instruments, instrument groups and their parameters from Instrument Service side. Note that there is no aggregation logic. Aggregated parameters should be configured manually (see InstrumentParametersUpdate and InstrumentGroupParametersUpdate classes for details).
