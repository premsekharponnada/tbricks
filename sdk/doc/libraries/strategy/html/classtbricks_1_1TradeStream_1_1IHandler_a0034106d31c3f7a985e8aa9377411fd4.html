pure virtual

Trade stream invalidation handler.

This method is called when an object previously present as a part of subscription in the handle stream update for the matched stream filter, now no longer fits filtering criteria. Fox example, a subscription was made for deleted Trade and all the deleted trades are received. If any such trade is un-deleted, invalidate method would be called to indicate that this trade no longer belongs to subscription request.
