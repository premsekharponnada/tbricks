Detailed Description

Filter for ReferencePrice::Stream to get reference prices in a date range including the boundaries.

To match all reference prices from the beginning of storage time until a given date or since a given date until the most recent date, set "from" or "until" date to empty respectively. Set both dates to empty to get all stored reference prices.

Deprecated:
Deprecated in 2.14.5 use ReferencePriceByDateFilter instead.
