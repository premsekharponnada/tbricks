Detailed Description

TradeByParentFilter matches all trades generated in response to orders directly created by a given strategy.

NOTE: The filter does not match indirect trades, that is, if strategy S1 creates strategy S2 and S2 sends an order that gets traded, the trade for this event will not match the filter. However, if order originates from S1, the trade would match the filter.
