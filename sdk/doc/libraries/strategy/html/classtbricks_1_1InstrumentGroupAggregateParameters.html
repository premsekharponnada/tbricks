Detailed Description

InstrumentGroupAggregateParameters is a merged view of group parameters and group's ascendant parameters. the instrument belongs to. That is, if group A has parameter PA and its parent group B that has parameter PB, then aggregate parameters for group A (unless they are excluded with manual parameter definition selection, see methods below) will contain both PA and PB.
