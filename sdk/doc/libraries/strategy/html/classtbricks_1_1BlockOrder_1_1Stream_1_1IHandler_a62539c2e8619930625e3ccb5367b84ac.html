virtual void HandleBlockOrderInvalidate ( const StreamIdentifier &  stream,

pure virtual

Order stream invalidation handler.

Called when an item previously seen on the stream stops to match stream filter, to signal that no strategy updates for this strategy instance will be available on the stream from now on, unless the instance changes.

For instance, if stream was open with a filter for all active block orders and an block order that previously matched the filter becomes filled, HandleOrderInvalidate() will be called for such instance.
