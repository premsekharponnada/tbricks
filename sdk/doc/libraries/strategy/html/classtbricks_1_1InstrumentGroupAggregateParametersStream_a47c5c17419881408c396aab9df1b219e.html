Open a stream.

Parameter "definitions" allows to select what specific instrument parameters updates the caller is interested in.

Parameter "snapshot_datetime" allows to get instument paramers as they were at a particular time in past. By default, "snapshot_datetime" is empty what means current parameters will be requested.

If stream is already opened or no callback interface is associated, the call has no effect.
