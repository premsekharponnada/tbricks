Detailed Description

StrategyInstanceParameters provide read/write access to a local strategy parameter container. Because every alteration of the container may result in a network traffic, this container is somewhat limited if compared to its mutable counterpart – massive alterations like Clear(), Swap() and such are not allowed here.
