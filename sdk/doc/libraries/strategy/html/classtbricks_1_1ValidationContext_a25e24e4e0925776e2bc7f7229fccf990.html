void RemoveItem ( const ParameterDefinition &  definition)

Remove item.

Permanently remove item from the context.

After this operation, it is illegal to use references to the item retrieved by GetItem() method with the same parameter or parameter definition.

If parameter has no corresponding item in the context, the call has no effect.
