◆ StartRecovery()

bool StartRecovery ( const InitializationReason &  reason)

Apps call this method when they need to start recovery process for their own orders. For example, after restart of the app.

Parameters
reasonReason to start recovery.
Returns
true The reason parameter is RECOVERED and other StartRecovery() is not running. This means there are may be orders to recover. Recovery will be performed.
false The reason parameter is not RECOVERED or other StartRecovery() is running.
