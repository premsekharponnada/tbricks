Detailed Description

InstrumentAggregateParameters is like InstrumentParameters, but unlike the latter it provides a merged view of parameters from instrument and groups the instrument belongs to. That is, if instrument A has parameter PA and it also member of group B that has parameter PB, then aggregate parameters for instrument A (unless they are excluded with manual parameter definition selection, see methods below) will contain both PA and PB.

Instruments can be thought of as being part of inheritance tree where ascendants are instrument groups, and each group might have a parent group, too, and aggregate parameters is the collection of all instrument and group parameters processed in top-down order.
