Detailed Description

Market Data Item object

Interface for providing market data. Market data item is associated with Instrument venue identifier and related parameters.

Market data item is passed in HandleSubscribe and HandleUnsubscribe methods. After HandleUnsubscribe result of MarketDataItem modification is unexpected.
